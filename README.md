# HK Mahjong (be) foreplay 🀄

A neon-drenched, interactive web application designed to teach the fundamentals of Hong Kong Old Style Mahjong. Built originally as a fun, practical way to teach your friends the ropes before sitting down at the physical table!

## 🌟 Features

* **Cyberpunk Neon Aesthetic:** A dark-mode, glowing interface that makes learning visually engaging.
* **Interactive Tile Dictionary:** Hover over Characters, Bamboo, Dots, Honors, and Bonus tiles to learn their names and meanings.
* **Step-by-Step Rules:** Clear breakdowns of sets (Chow, Pong, Kong), the 14-tile winning rule, and Claim Priority.
* **Winning Patterns Guide:** Visual examples of scoring hands, from basic minimum-Fan hands to legendary 64-Fan limits (like *Thirteen Orphans* and *All Honors*).
* **Interactive "Build The Hand" Builder:** A dynamic practice mode that generates random scenarios where players must identify the missing tile to complete a waiting hand.
* **Final Exam Quiz:** A randomized, scored quiz testing game setup, rules, and scoring knowledge.

## 🛠 Tech Stack

This entire application is lightweight and runs entirely in the browser using a single file. No build steps or package managers required.

* **React 18** (loaded via CDN)
* **Tailwind CSS** (for styling and neon effects)
* **Babel** (for in-browser JSX compilation)
* **HTML5 / CSS3** ## 🚀 How to Run

Because the app is entirely client-side, getting it running is instant:

**Option 1: Play Locally**
1. Clone or download this repository.
2. Double-click the `index.html` file to open it in any modern web browser (Chrome, Safari, Firefox, etc.).


## 💡 About

Learning Mahjong can be intimidating with its deep rulebook, countless variations, and fast-paced table etiquette. This project breaks down the barrier to entry, offering an interactive and pressure-free environment to master the basics.
